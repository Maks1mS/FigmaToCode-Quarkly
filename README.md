To transfer a design from Figma to Quarkly you need:

1. Start Plugin
2. Select Frame
4. Copy the resulting code
5. Paste the code into your project page in Quarkly

A fork of [Figma To Code](https://github.com/bernaferrari/FigmaToCode)
